To enable Google authentication in this app, use these steps:

1. Create a Google Cloud project
   - Go to Google Cloud Console.
   - Create a new project or select an existing one.

2. Enable the Google OAuth consent screen
   - In APIs & Services, open “OAuth consent screen”.
   - Choose External.
   - Fill in the app name, support email, and developer contact.
   - Save and continue.

3. Create OAuth credentials
   - Go to APIs & Services → Credentials.
   - Click Create Credentials → OAuth client ID.
   - Application type: Web application.
   - Add your app URL under Authorized redirect URIs:
     - https://localhost:5001/signin-google
     - or the HTTPS URL you use when running the app.
   - Save the generated Client ID and Client Secret.

4. Put the credentials into the app
   - Open [Hotel-Mgt.Web/appsettings.json](Hotel-Mgt.Web/appsettings.json).
   - Set:
     - Authentication:Google:ClientId
     - Authentication:Google:ClientSecret
   - If you are debugging locally, you can also add them to [Hotel-Mgt.Web/appsettings.Development.json](Hotel-Mgt.Web/appsettings.Development.json).

5. Restart the app
   - Run:
     dotnet run
   - Then open the login page and use “Continue with Google”.

Important note:
- The Google button is already wired in the login page, but it will only work once the real Google credentials are provided in the settings.
