# Ocjenjivanje

Uvjet za potpis: minimalno 50% bodova na lab. vježbama. Vježbe nose 30 bodova.

## Projekt - specifikacija

Predaja na rokovima. Ne mora svaki kriterij biti ispunjen.

| Kriterij | Bodovi |
|---|---|
| Deploy na cloud provider (Google, Azure) ili Virtual Machine | 3 |
| Kreiranje testova za sve API endpointe (Playwright scenarij 10 koraka +3 extra boda) | 2 |
| AI integracija: unos podataka putem AI upita i sl. | 3 |
| Global search - mogućnost pretrage izbornika i stranica i podataka | 2 |
| Implementacija logging mehanizma (file ili API) | 2 |
| Responsive - mobile/web UI | 2 |
| CRUD mora raditi bez grešaka | 2 |
| Expose MCP i pristup kroz agentic IDE | 2 |
| Okvirni dojam funkcionalnosti aplikacije (da nema 'crasheva') | 12 |
| Verifikacija razumijevanja koda (usmeno ispitivanje) | 40 |
| **Ukupno** | **70** |
